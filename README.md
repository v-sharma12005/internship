# Internship Project

## Overview
This project is an interactive car animation website built with modern web technologies. It provides hands-on experience in web development, animation, and scroll-triggered effects using GSAP (GreenSock Animation Platform). The primary goal is to create an engaging user experience with smooth animations and visual effects.

## Features
- **Headline Animation**: Split-text animation with staggered character-by-character reveal effect on page load
- **Scroll-Based Car Animation**: Dynamic car image that moves and scales as users scroll down the page
- **Statistics Display**: Animated stat items that fade in with smooth transitions
- **Responsive Design**: Clean and modern UI with smooth visual feedback

## Project Structure
```
internship/
├── index.html      # Main HTML structure and layout
├── style.css       # Styling and responsive design
├── script.js       # Animation logic using GSAP and ScrollTrigger
├── car.png         # Car image asset for animations
└── README.md       # Project documentation
```

## Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling and responsive layout
- **JavaScript**: Dynamic functionality and interactions
- **GSAP (GreenSock Animation Platform)**: Professional-grade animation library
- **ScrollTrigger Plugin**: Scroll-based animation triggers

## Installation
To set up this project locally:
1. Clone the repository using `git clone https://github.com/v-sharma12005/internship.git`
2. Navigate to the project directory: `cd internship`
3. Open `index.html` in your web browser
4. No additional dependencies are required - all libraries are loaded via CDN

## File Descriptions

### index.html
- Defines the page structure with hero section containing the title, statistics, and car image
- Imports GSAP library and ScrollTrigger plugin from CDN
- Links to external CSS and JavaScript files

### style.css
- Provides styling for all page elements including typography, layout, and spacing
- Implements responsive design for different screen sizes
- Defines animations and visual effects

### script.js
- Implements headline animation by splitting text into individual character spans
- Creates staggered animation effects for better visual appeal
- Implements scroll-triggered car image animations using ScrollTrigger
- Animates stat items with fade-in and slide-up effects

### car.png
- High-quality car image asset used in the hero section
- Dimensions: optimized for web display

## Usage
1. Open the `index.html` file in a modern web browser
2. Observe the headline animation on page load
3. Scroll down to see the car image animation and stat items appear with smooth transitions
4. The animations are triggered automatically based on scroll position
